Новый курс «Основы HTML 5.0»

Курс «Основы HTML 5.0» предназначен для широкого круга пользователей, желающих познакомиться с языком гипертекстовой разметки HTML. По окончании курса слушатели смогут самостоятельно создавать web-страницы, содержащие все необходимые компоненты: текстовое наполнение, гиперссылки, графическое наполнение, создавать полноценный сайт путем определения иерархии взаимодействия web-страниц между собой, создавать интерактивные формы для взаимодействия пользователя с web-сервером, а также использовать технологию фреймов.
